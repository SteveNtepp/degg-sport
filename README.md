# degg-sport
Plan Directeur Stratégique pour la Super-App Dëgg-Sport de l'ONADS Sénégal. Document de conseil Big4 présentant l'architecture de transformation digitale antidopage (4 modules intégrés), la feuille de route JOJ Dakar 2026, l'analyse financière hybride et la matrice des risques. Livrable HTML/CSS print-ready.
